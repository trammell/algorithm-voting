# Description #

This project documents and implements a variety of voting systems, including:

  * "single-winner"
    * Plurality
    * IRV
    * Approval
    * Range Voting
    * Condorcet variations
  * "multiple-winner"
    * D'Hondt
    * Highest averages
    * Sortition

The implementation language is Perl, with a focus on thorough testing and high code coverage.
